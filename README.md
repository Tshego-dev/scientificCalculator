# Tshego's Scientific Calculator

A modern, user-friendly scientific calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations (+, -, ×, ÷)
- Scientific functions (sin, cos, tan, log, ln, √, x², !)
- Mathematical constants (π, e)
- Memory functions (MC, MR, M+, M-)
- Power operator (^)
- Clean, modern UI
